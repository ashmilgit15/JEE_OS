## Project Overview

**JEE OS** — A Next.js 16.2.7 + React 19.2.4 application for JEE (Joint Entrance Examination) preparation. Features include AI tutor/coach, syllabus tracking, question bank, analytics, revisions planner, and study logging.

**Stack:** Next.js App Router, Tailwind v4 (CSS-first), `@base-ui/react` (not Radix), Supabase SSR, Hack Club AI Proxy, Tavily search.

## Architecture

### Routing & Structure
- **Single-page App Router** — All routes under `src/app/`. No monorepo, no separate packages.
- **Route groups** organize features: `(auth)`, `(dashboard)`, `(marketing)`, etc.

### State Management
- **Custom React Context + `useReducer`** in `src/store/index.tsx`
- Persisted to `localStorage` under key `jee-os-state`
- Actions defined in `src/store/actions.ts`, types in `src/types/index.ts`

### UI Components
- **`@base-ui/react`** (NOT Radix/shadcn) — generated by `shadcn` using `base-nova` style
- All reusable components in `src/components/ui/`
- Components use `render` prop for polymorphic composition (not `asChild`)

### Authentication
- **Supabase SSR** via `@supabase/ssr` (not `@supabase/supabase-js` directly)
- Clients: `src/utils/supabase/client.ts` (browser), `server.ts` (server), `middleware.ts` (route guard)
- Env var: `NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY` (NOT `ANON_KEY` — handoff doc is wrong)

### Styling
- **Tailwind v4 CSS-first** — `@theme` in `globals.css`, no `tailwind.config.js`
- Dark-first design, CSS variables in `oklch()`
- Custom Button sizes: `xs`, `icon-xs`, `icon-sm`, `icon-lg` (from `class-variance-authority`)

### API
- `/api/chat/route.ts` — Streams NDJSON for AI tutor/coach
- Falls back to hardcoded smart fallback if Hack Club API key missing
- External: Hack Club AI Proxy (chat completions), Tavily (web search)

### Middleware
- `src/proxy.ts` (renamed from `middleware.ts` per Next.js 16.2.7 deprecation)
- Handles route guard / session sync

## Known Issues (Do Not Re-introduce)

### Lint Errors (7)
- `analytics/page.tsx:898,906` — 4× `react/no-unescaped-entities` (`"` should be escaped)
- `api/chat/route.ts:264,281,367` — 3× `prefer-const` (`useFallback`, `currentMessages`, `topicNameCandidate`)

### Lint Warnings (88)
- Many unused imports (`@typescript-eslint/no-unused-vars`)
- `revisions/page.tsx:225` — `new Date()` in `useMemo` deps breaks memoization
- `syllabus/page.tsx:394` — `state` as `useMemo` dep causes full re-compute
- `supabase/middleware.ts:15` — `supabase` variable created but never used

### Logic Bugs
- `log/page.tsx:597` — Tabs use `defaultValue` (uncontrolled) but `setSelectedSubject` tries to control programmatically → UI/state desync
- `tutor/page.tsx:627` & `FloatingAICopilot.tsx` — `new URL(result.url).hostname` crashes if URL malformed (no try/catch)
- `api/chat/route.ts` — Uses `let` for never-reassigned vars
- `store/index.tsx` — `isToday`, `startOfDay` imported from `date-fns` but unused
- `handoff.md` documents `NEXT_PUBLIC_SUPABASE_ANON_KEY` but code uses `PUBLISHABLE_KEY`

### Tailwind v4 Gotchas
- No `tailwind.config.js` — config is in `globals.css` via `@theme`
- `h-4.5` / `w-4.5` classes in some pages are INVALID (not in v4 spacing scale)

### Component API Differences from Standard shadcn/Radix
- `@base-ui/react` uses `render` prop (not `asChild`) — see `Sidebar.tsx:118`, `sheet.tsx:65`
- Tooltip: `TooltipPrimitive.Popup` + `TooltipPrimitive.Positioner` (not `Content`+`Trigger`)
- Sheet: based on `@base-ui/react/dialog` (not `@radix-ui/react-dialog`)

### Supabase Caveats
- Server client (`server.ts`) requires `cookieStore` from `next/headers`
- `middleware.ts` returns `supabaseResponse` but never uses the `supabase` client it creates
